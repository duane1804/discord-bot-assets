# Discord Bot Assets

Game icons for AllInOne Discord RPG Bot, served via jsDelivr CDN.

Icons from [game-icons.net](https://game-icons.net) — CC BY 3.0
Authors: Lorc, Delapouite (https://game-icons.net)

## CDN URL
```
https://cdn.jsdelivr.net/gh/duane1804/discord-bot-assets@main/
```

## Structure
- `lpc/weapons/` — Sword icons (fire, ice, lightning, earth, dark variants)
- `lpc/armor/` — Chest, helmet, boots
- `lpc/items/` — Ring, potion, ore, gem, ticket
- `lpc/categories/` — Fallback category icons
- `lpc/classes/` — Player class icons (warrior, mage, rogue, paladin)
- `lpc/monsters/` — Monster icons (slime, skeleton, goblin, wolf, dragon, boss)
